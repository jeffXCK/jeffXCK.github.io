git分支相关命令

发表于 2019-09-29 分类于 git 阅读次数：
本文字数： 230 阅读时长 ≈ 1 分钟

创建分支：

  • git branch <分支名>
  • git branch -v 查看分支

切换分支：

  • git checkout <分支名>
  • 一步完成： git checkout -b <分支名>

合并分支：

  • 先切换到主干 git checkout master
  • git merge <分支名>

删除分支

  • 先切换到主干 git checkout master
  • git branch -D <分支名>
